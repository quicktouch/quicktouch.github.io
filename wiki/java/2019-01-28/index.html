(JB)idea添加项目依赖、以及打包问题

wiki  /  java  / idea 2019-01-28

单窗口多项目支持

  1. file -> project structure 或 cmd + ;

  2. 点击+,根据实际情况选择import Module或new Module

项目间依赖

类似的,选中右下的加号,配置项目间依赖。

打包

B依赖A,直接打包B可能会报错: ‘Could not resolve dependencies for project’,处理方法: 被依赖的项目先进执行install,主项目再进行package
