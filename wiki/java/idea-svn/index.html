(JB)IDEA导入svn项目二三谈

wiki  /  java  / IDEA 2018-11-06

1). idea svn certificate issued for a different hostname, issuer is not trusted

命令行里添加信任:

svn ls 资源库地址

随后按提示选择p

2). svn: e175009: the xml response contains invalid xml

问题原因: 地址不合法。

直接从浏览器地址栏复制的会有问题,改为不带!

正确的地址：

`https://10.10.50.198:198/svn/xxx资源库

而不是输入浏览器后的坑爹地址：

https://10.10.50.198:198/!/#xxx资源库/

或者路径: user@svn:路径
