macOS已安装软件一览

wiki  /  mac 2018-12-08

准备重装系统,列举一下已安装的软件以作不时之需。

文本

Reeder.app 订阅 Quiver.app markdown编辑器 Evernote.app Ulysses.app 老牌markdown编辑器,代码风格不怎么喜欢 GitBook Editor.app gitbook编辑工具,很一般 MWeb.app markdown编辑器,文件过多有卡顿问题 Microsoft OneNote.app 好用但笨重 MacDown.app

IDE or 编辑器

PyCharm.app RubyMine.app CodeRunner.app 代码段play ground Eclipse.app 被idea秒杀 wechatwebdevtools.app AppCode.app 比xcode编码体验好 Android Studio.app PhpStorm.app Sublime Text.app 编辑器颜值担当 Unity HBuilder.app 代码提示很全 IntelliJ IDEA.app JAVA开发top1 Visual Studio Code.app 万用,但是各种插件有点恶心 Xcode.app WebStorm.app Brackets.app

开发相关

Easy APNs Provider.app APNS调试工具 Knuff.app APNS推送？ SmartOTA.app 有时候会用。狐总出品 SourceTree.app Impactor.app 越狱签名工具 ida 2.app 逆向工具,没用过几次 UltraEdit.app 可以编辑文本、十六进制、ASCII 码 Charles.app 神器 Reveal.app 神器 postman 神器 iOS Images Extractor.app 提取ipa资源 JSONExport.app json -> 模型类代码,个人喜欢在线工具 Yummy FTP.app mac上最好用的FTP软件 MediaInfo.app 分析多媒体文件格式 Paw.app 不如postman

设计相关

PxCook.app Sketch.app Adobe Photoshop CC 2017

社交娱乐

QQ.app WeChat.app IINA.app 开源播放器 VLC.app 播放器 MPlayerX.app 播放器 iTVSeries 看美剧,github:http://itvseries.github.io/iTVSeries

工具

AirDroid.app 安卓手机投屏还行
1Password.app 密码管理神器 QuickTime Player.app Cornerstone.app svn工具,烂,经常卡死 DaisyDisk.app 电脑存储情况分析软件
AppDelete.app 清理删除软件使用
Clearview.app 阅读器
Citrix Workspace.app 虚拟桌面插件 EasyConnect.app CleanMyMac 3.app 清理工具,一般不清理 SSH Shell.app 比较方便的是可以记录主机ip啥的,但是我用alias不得了 Dash.app 文档管理 pgAdmin 4.app postgresql数据库工具 Sequel Pro.app 数据库管理,被navicat秒杀
Navicat Premium.app
ShadowsocksX-NG.app 神器
Downie.app 下载工具,个人很少用到
The Unarchiver.app 解压 Dropbox.app 国内环境有问题 Hex Fiend.app 二进制对比工具 Tuxera Disk Manager.app Tuxera NTFS U图床.app 开源的图床工具 KeyCue.app 神器,可以提示快捷键 Karabiner.app 神器,改键 HandShaker.app 锤子家的手机工具 VirtualBox.app 虚拟机用到的,好像 Jason.app json解析工具,不如在线工具好用 LICEcap.app 动图录屏 MAMP PRO mamp环境懒人版 Alfred 3.app 神器不解释
OhMyStar.app github管理star的软件,不过买了之后没用过几次 PDMan.app 数据库设计,简洁大方 Parallels Desktop.app 虚拟机,算是mac上最好用的了吧 Apple Configurator 2.app 管理手机软件,itunes废了
Aria2GUI.app 百度网盘下载助手,长时间不用忘了咋用了
Axure RP 8.app
PicU.app 图床软件,收费的,已经卸了 iPic.app 图床

效率

Things.app 待办清单 FocusTime.app 类似番茄工作法 Understand.app 源码阅读 HandBrake.app 老牌的视频转换工具 Inboard.app 收集图片灵感,图片碎片分类 XMind.app 脑图,多平台兼容 MindNode.app 脑图 wps.app WPS系列 Microsoft Remote Desktop.app 桌面远程连接 iTerm.app 神器 智图.app 图片压缩(“无”损)

浏览器

Google Chrome.app Firefox.app
